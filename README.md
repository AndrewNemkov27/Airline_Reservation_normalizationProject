# Airline_Reservation_normalizationProject
Airline reservation system and database normalization project. Includes data processing, functional dependency analysis, normalization (1NF to BCNF), SQL script generation, MySQL database population, and an interactive query interface built in Python.

## Additional Notes

### Files Not Included
Due to file size limitations, the following files are not included in this repository:

- *Original datasets*: `PNR.xml`, `iata.txt`
- *Intermediate datasets*: `PNR.csv` (Part A), `PNR_full.csv` (Part B)
- *Normalized table CSVs*:  
  `Seats_norm.csv`, `Reservations_norm.csv`, `Flights_norm.csv`, `Passengers_norm.csv`

These files can be regenerated by running the provided Python scripts.
